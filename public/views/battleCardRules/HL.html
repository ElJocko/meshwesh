Rules Change

Prevents one stand from following up after one combat.

Can only be applied to troop types marked in the army list with this ability, usually Elite Foot.

Timing

The choice to use this ability is made after the combat result is computed and applied, before the next combat.

This is an event card and is expended when used; flip the card upside down to mark that it is no longer available for use in the current battle.

Cost

0 points for the first such battle card; 1 point each for any subsequent ones.
