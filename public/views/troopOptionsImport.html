Import Army List Troop Options
Use this page to import army list troop options data. The data should be in a csv file. Do not include spaces before or after the data (any spaces will be included in the imported data). The first row should be a header row in the following format:
listId,sublistId,troopOptionOrder,startDate,endDate,general,minMax,troopEntries,description,core
The description field is also the army list status. Imported data will replace the troop options for all army lists included in the file. Army lists not in the file will be unaffected.
