Import Troop Types
Use this page to import troop type data. The data should be in a csv file, using a vertical bar as a delimiter. Do not include spaces before or after the data (any spaces will be included in the imported data). The first row should be a header row in the following format:
permanentCode|displayName|displayCode|cost|category|description
Imported troop types will replace the current troop types.
